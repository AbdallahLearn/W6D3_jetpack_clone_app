# UI Clone using Jetpack Compose

## Description

This project is a UI clone built using **Jetpack Compose**, focusing on modern Android UI development. It includes animations, multiple screen handling, and localization for **English and Arabic**. The app is designed to be responsive, adapting seamlessly to both **mobile and tablet** devices.

### Key Features:
- **Cloned UI screens** with smooth user interactions.
- **Animations** for enhanced user experience.
- **Multi-language support** (English & Arabic).
- **Optimized layout** for mobile and tablet devices.
